# IMS GPA Calculator
## Description

This is a web-based GPA Calculator built using HTML, CSS, and JavaScript. The IMS GPA Calculator allows users to input their course grades and credit hours to calculate their Grade Point Average (GPA) instantly. It provides a simple and user-friendly interface for students to track their academic performance efficiently.
