# IMS GPA Calculator

## Project Description
The IMS GPA Calculator is a versatile tool designed to assist students in computing their Grade Point Averages (GPA) based on their course grades and credits. This project streamlines the calculation process, providing an easy-to-use interface for accurate GPA computation.

## Features
- **User-Friendly Interface**: Simple and intuitive UI for easy navigation.
- **GPA Calculation**: Calculates GPA based on grades and credit hours.
- **Support for Multiple Semesters**: Allows input and calculation for multiple semesters.
- **Export Results**: Ability to export GPA results to various formats (PDF, Excel).

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js (if applicable)
- **Database**: MongoDB (if applicable)
- **Version Control**: Git

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/AbuBakar-Sadiq-ai/IMS-GPA-Calculator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd IMS-GPA-Calculator
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage Guide
1. Enter your course grades and credit hours into the provided fields.
2. Click the 'Calculate GPA' button.
3. Review your GPA results displayed on the screen.
4. Optionally, export your GPA results using the export feature.
